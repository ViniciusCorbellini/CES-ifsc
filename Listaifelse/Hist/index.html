História do Java

Java, concebido por James Gosling em 1995 durante seu tempo na Sun Microsystems, não começou como a linguagem de programação powerhouse que é hoje, embora tenha ganhado popularidade rapidamente após seu lançamento.

O desenvolvimento inicial do que se tornaria o Java teve início na Sun Microsystems em 1991. Originalmente chamado de Oak, o projeto visava inicialmente a televisão interativa. No entanto, quando se constatou que o Oak era muito avançado para a tecnologia de cabo digital disponível na época, Gosling e sua equipe redirecionaram seus esforços para criar uma linguagem de programação e rebatizaram o projeto como Java, em homenagem a um tipo de café da Indonésia. Gosling viu o Java como uma oportunidade para abordar problemas que ele antevia com linguagens de programação menos portáveis à medida que mais dispositivos se tornavam conectados em rede.

O Java foi projetado com um estilo de sintaxe semelhante à da linguagem de programação C++, tornando-se familiar para os programadores ao começarem a usá-lo. Embutido com o lema "escreva uma vez, execute em qualquer lugar", um programador poderia escrever código Java para uma plataforma e executá-lo em qualquer outra plataforma que tivesse um interpretador Java (ou seja, a Máquina Virtual Java) instalado. Com a ascensão da Internet e a disseminação de novos dispositivos digitais na década de 1990, o Java foi rapidamente adotado pelos desenvolvedores como uma linguagem de programação verdadeiramente multiplataforma.

A primeira versão pública do Java, o Java 1.0, foi lançada em 1996. Em cinco anos, atraiu 2,5 milhões de desenvolvedores em todo o mundo. Atualmente, o Java alimenta uma ampla gama de sistemas, desde o sistema operacional móvel Android até software empresarial.
